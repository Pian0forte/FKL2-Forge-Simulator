# FKL2-Forge-Simulator
A browser-based, high-fidelity crafting simulator that replicates the Recast Station mechanics from Fort Knight Legend 2. This tool allows players to experiment with item affix rolling, lock mechanics, and stat maximization without burning through in-game resources.

🚀 Live Demo
[https://philosophical-sapphire-rx7lxd7xrj.edgeone.app/]

✨ Features
Affix Recasting System: Simulates exact Action-RPG crafting logic by rolling Prefixes and Suffixes simultaneously.

Intelligent Logic Modules: * RECAST ONE: Re-rolls a random unlocked slot.

RECAST LOWEST: Targets and optimizes your weakest tier stats first.

Authentic Pool Distributions: Features actual simulated weightings for generic stats, exclusive skill-tier limitations (T3/T6 for All Skills), and weapon-specific modifiers.

Complete Sandbox Tools:

🎲 Fresh Drop: Completely rolls a new item with random Prefixes, Suffixes, and a Mythic/Implicit passive.

⚠️ Corrupt Card: Toggles a corruption banner to simulate end-game gear bricking or highlighting.

📝 Inline Text Editing: Manually edit item names, levels, and base physical stats directly on the tooltip display.

Immersive Forge SFX: Integrated dynamic Web Audio API synthesizer that plays an authentic metallic forge sound with every hammer strike.
